## 🧠 Key Skills I'll Gain
- Real-time operating system concepts
- FreeRTOS task scheduling and configuration
- Inter-task communication and synchronization
- Priority inversion and deadlock handling

## 🔗 How It Connects to VR Hardware
VR systems have strict latency budgets — motion-to-photon latency must stay under 20ms to prevent discomfort. Real-time scheduling is what makes that possible at the firmware level.
