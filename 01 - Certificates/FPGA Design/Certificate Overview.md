# FPGA Design for Embedded Systems Specialization

## 📌 Why I'm Doing This
> To gain hardware-level design skills in FPGA — the programmable hardware layer used in VR headset prototyping, display processing, and sensor fusion pipelines.

## 🧠 Key Skills I'll Gain
- VHDL and Verilog hardware description languages
- FPGA architecture and design flow
- Softcore processor integration
- IP block design and reuse
- Hardware simulation and timing analysis

---

## 🔗 How It Connects to VR Hardware
FPGAs are used in VR hardware for low-latency display processing, sensor data fusion, and prototyping custom silicon before tape-out. This specialization builds the HDL skills needed to work at that layer.
